### 🛠️ What I do

I build high-throughput backend systems in **native PHP** and break things (ethically) to make them harder to break. My work sits at the intersection of **backend engineering**, **anti-fraud / bot-detection**, and **defensive security** — vulnerability research, WAF rule design, authentication-bypass analysis, and server hardening.

- 🤖 Founder &amp; engineer of **[stopbot.net](https://stopbot.net)** — a bot-detection &amp; anti-fraud SaaS serving **~10M requests/day** on native PHP + MySQL.
- 🛡️ Security research — CVE investigation, auth-bypass analysis, WAF rules, cPanel / Linux hardening.
- ⚙️ Self-managed infrastructure — VPS, Nginx/Apache, MySQL tuning, Redis, MikroTik network plumbing.
- 🧪 Leveling up in **Go**, with side trips into **Python** and **Rust**.

### 🚀 Currently working on

Scaling **stopbot.net**'s data layer under load: a **Redis verdict cache** (epoch-based invalidation + stampede protection), **ProxySQL** for connection multiplexing and read/write split, composite indexing, read replicas, and async analytics offload to **ClickHouse** via Redis Streams. Migration follows a **strangler-fig** pattern — hot path moving toward Go, session panel staying PHP.
